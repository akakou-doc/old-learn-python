# 12. while 文、for 文、continue、break

## while 文

while 文はインデントされているところの処理を、条件式が True の間処理を実行する構文です。

```py
# 無限にwan!と唱え続ける
while True:
    print('wan!')
```

```py
# 一度も実行しない
spam = False
while spam:
    print('spam')
```

```py
# 100以上を数えるまで実行する
count = 0
while count < 100:
    count = count + 1
    print(count)
```

## for 文

for 文を用いると、リスト等から 1 つずつデータを取り出して、インデントされているところの処理を実行することができます。

```py
spam = [1,2,3,4,5]

for ham in spam:
    print(ham)
```

### 演習 3

```py
numbers = [3, 4, 5, 3, 42]
```

という配列があります。この中の数字を全て表示して下さい。
但し、表示する数字が 3 の倍数だった場合、`!`を数字の右ににつけて下さい。

## continue

continue を使うと、処理の途中で、次のループの処理に入ることができます。

```py
count = 0

while True:
    input()
    count = count + 1

    if count % 2 == 0:
        # もし偶数なら表示せず
        # 次のループに向かう
        continue

    print(count)
```

## break

break を使うと、処理の途中で、ループを終了することができます。

```py
count = 0

while True:
    count = count + 1

    if count >= 100:
        # もしcountが100より
        # 大きくなったらループを終了する
        break

    print(count)

print('end !')
```
