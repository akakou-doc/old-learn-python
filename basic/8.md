# 標準で用意される関数

Python では標準で、様々な関数が用意されています。

## print 関数

print 関数では**引数**の値を画面に表示します。

```py
# 0 が表示される
print(0)

# hello world が表示される
print('hello world')
```

## input 関数

一行入力を受け取って返り値として返します。

```py
# 1行読み取って表示する
print(input())

# 2行目を読み取って、その値を変数 line に代入する
line = input()

# 変数 line の値を表示する。
print('2回目: ' + line)

---
あああ
いいい
---
```
