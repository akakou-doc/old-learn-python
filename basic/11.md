# 辞書（dict）

辞書とは複数の値を格納できる値で、key となる値と value となる値を紐付けることによって操作をします。
この値を dict 型の値と呼びます。

例えば`{'name': 'akakou', 'age': 19, 'is_human': False}`という辞書があったら、key と value のひも付きは以下の表の用になります。

|  key  |  'name'  | 'age' | 'is_human' |
| :---: | :------: | :---: | :--------: |
| value | 'akakou' |  19   |   False    |
